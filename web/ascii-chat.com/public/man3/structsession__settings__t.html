Detailed Description

Session settings for transmission between peers.

Contains display and rendering configuration that can be synchronized between session participants. Uses fixed-size fields for deterministic serialization.

Note
Struct is designed for binary serialization with network byte order.
Reserved bytes allow future extension without breaking compatibility.
