Scalar truecolor halfblock renderer.

Parameters
rgbInput RGB image data (RGB24, 3 bytes per pixel)
widthImage width in pixels
heightImage height in pixels
stride_bytesBytes per row (if 0, calculated as width * 3)
Returns
Malloc'd NUL-terminated ANSI escape string (caller must free)

Renders an RGB image using halfblock characters with truecolor foreground and background colors. Processes 2 source rows per output line:

  • Top pixel: foreground color for halfblock
  • Bottom pixel: background color for halfblock
  • Last row (odd height): duplicates top row as bottom

Uses RLE optimization for repeated halfblocks and detects transparent areas (fully black pixels) which are rendered as spaces instead of halfblocks.
