Detailed Description

Connection state machine and attempt logic for TCP/WebSocket clients.

Implements direct TCP and WebSocket connections for client mode. Manages connection state transitions, timeouts, and transport lifecycle.

Features:

  • Direct TCP connection to server
  • WebSocket connection support
  • Timeout management
  • Crypto handshake integration
  • ACIP transport creation
  • Proper resource cleanup

Integration Points:

  • Called from src/client/main.c connection loop
  • Returns active transport when connection succeeds
