Detailed Description

Implementation of atomic operations abstraction.

Date
March 2026

Provides _impl functions for atomic operations. In debug builds, these functions are wrapped by the typed macros to include debug hooks. In release builds, the macros call these directly for zero overhead.

Design: Uses raw C11 atomic operations internally, never calls back into the atomic_t system (no recursion). Debug hooks are called after the atomic operation completes.
